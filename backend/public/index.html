        Tips for saving tokens

        • Summarize buttons reuse the cached review.summary from the first score — repeat clicks are free.
        • Filter queries like "backend devs with 2+ years" are answered with structured SQL before any AI call.
        • Embeddings run locally (Transformers.js), so semantic search is free and offline.
        • Re-scoring a previously-seen email attachment is deduped by (email_id, attachment_id).
